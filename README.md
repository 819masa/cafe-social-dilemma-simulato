## 概要

カフェにおける長時間滞在問題を社会的ジレンマとして分析し、「寄付×席予約権ハイブリッド型メタ報酬システム」の効果をエージェントベースモデルで検証するシミュレーターです。

ブラウザ上で動作するスタンドアロンのHTMLファイルとして実装されており、インターネット接続があればそのまま開いて使えます。

---

## 背景・理論

本シミュレーターは以下の講義理論に基づいて設計されています。

- **公共財ゲーム**：カフェの席を共有資源とみなし、個人の利得と集団の利得の対立を定式化

- **メタ報酬ゲーム**（Axelrod 1986）：罰則ではなく報酬によって協調を促進する仕組みの検証

- **TFT（Tit for Tat）戦略**：繰り返しゲームにおける互恵的協調の制度化

- **社会的ワクチン**：少数の非協調者の存在が系全体の協調を安定化させる効果

## シミュレーターの機能

- **システムあり／なしの切り替え**：ハイブリッド型メタ報酬システムの有無を比較

- **パラメータ調整**：総席数・C型割合・D型割合・来客頻度・固定費をスライダーで変更

- **リアルタイム可視化**：カフェ平面図（座席グリッド）で各エージェントの状態を色分け表示

- **時系列グラフ**：混雑率・協調退店数・C型占有数・累積利益の推移をリアルタイム描画

- **イベントログ**：各ステップの状態をテキストで記録

## 使い方

```
1. cafe_simulator.html をブラウザで開く
2. 「システムなし」または「ハイブリッドシステムあり」を選択
3. パラメータをスライダーで調整
4. 「▶ 開始」ボタンでシミュレーション開始
5. 「リセット」で初期状態に戻す
```

**推奨実験：**
同一パラメータ（C型30%・来客頻度3・固定費400）でシステムあり／なしを比較すると、混雑率・累積利益・F型離脱数の差が明確に現れます。
